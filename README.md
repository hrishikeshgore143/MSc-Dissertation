# MSc Dissertation

## Student Academic Performance Prediction Using Machine Learning

### Overview
This project was developed as part of my MSc Dissertation at GISMA University of Applied Sciences.

The project predicts student academic performance using several machine learning algorithms and provides predictions through a Streamlit web application.

## Features

- Student performance prediction
- Streamlit web interface
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- SHAP Explainability

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- SHAP
- Matplotlib

## Repository Contents

- `master_thesis.py` – Main application
- `README.md` – Project documentation

## How to Run

1. Install the required libraries

```bash
pip install -r requirements.txt
```

2. Run the application

```bash
streamlit run master_thesis.py
```
